Pengaturan Sync Google Sheets

Integrasi opsional — data survey otomatis dikirim ke Google Sheet setelah submit. Kalau kosong, data tetap tersimpan di device.

📋 Setup Apps Script (klik untuk expand)

Langkah setup di Google Sheets:

  1. Buat Google Sheet baru → beri nama (misal "Survey Churn Jabeka 2026")
  2. Menu Extensions → Apps Script
  3. Hapus kode default, paste kode di bawah
  4. Ubah SECRET agar match dengan field di atas
  5. Simpan (💾), lalu klik Deploy → New Deployment
  6. Type: Web app · Execute as: Me · Access: Anyone
  7. Copy Web App URL → paste ke field di atas
  8. Klik Test Sync di bawah untuk verifikasi
